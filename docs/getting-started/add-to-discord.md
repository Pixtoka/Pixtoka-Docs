# Add to Discord

Adding Pixtoka to your Discord server can be done using the following link :

| Add Pixtoka to Discord ➡️ | https://pixtoka.xyz/invite |
| ------------------------- | -------------------------- |

:::note

You must first give permissions to Pixtoka to start using it into your server. Head to the [permissions](/getting-started/permissions) page.

:::
